Network UART Console Summary
File	Purpose	Key Features
UARTConsole.hpp	UART console header	Types, structures, commands, public API
UARTConsole.cpp	UART console implementation	Command processing, color output, logging, built-in commands
Key Features

    Command System:

        Register custom commands with handlers

        Built-in commands (help, clear, echo, prompt, loglevel, status)

        Command parsing with argument support

        Command help system

    Output Features:

        Colored text support

        Logging with levels (DEBUG, INFO, WARNING, ERROR, CRITICAL)

        Formatted output

        Clear screen

    Interactive Features:

        Prompt with echo

        Backspace support

        Ctrl+C (interrupt)

        Ctrl+L (clear screen)

    Integration:

        Callback for log messages

        Callback for commands

        Thread-safe operation

Usage Example
cpp

// Create and initialize console
UARTConsole console;
console.initialize("/dev/ttyAMA0", 115200);

// Set prompt
console.setPrompt("record> ");

// Register custom command
console.registerCommand("record", "Start recording", 
    [](const std::vector<std::string>& args) -> std::string {
        // Start recording logic
        return "Recording started";
    });

console.registerCommand("status", "Show status",
    [](const std::vector<std::string>& args) -> std::string {
        return "System: Online, Recording: Idle";
    });

// Enable console
console.enable();

// Log messages
console.log(UARTConsole::LogLevel::INFO, "System started");

// ... application runs ...

// Disable and shutdown
console.disable();
console.shutdown();
